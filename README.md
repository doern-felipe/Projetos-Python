# Projetos-Python
Este repositório reúne diversos projetos e exercícios desenvolvidos em Python, organizados por cursos, práticas pessoais e experimentos.
O objetivo é consolidar o aprendizado da linguagem, registrar a evolução ao longo dos estudos e servir como portfólio de projetos.

# Objetivo
- Documentar a trajetória de aprendizado em Python.
- Centralizar projetos de diferentes cursos e práticas.
- Facilitar a consulta e reutilização de códigos.
- Servir como referência para futuros projetos.
